# Lumina: Firefly Science 🌌💡

A deluxe, high-contrast exploration of the biological mystery behind fireflies and their "cold light."

## Features

- **Responsive Design:** Optimized for both mobile and desktop.
- **Modern UI:** Dark theme with high-contrast accents (Yellow/Lime).
- **Interactive Background:** Real-time canvas-based firefly particles with unique pulse rates.
- **Scroll Reveal:** Smooth transitions and reveals as you explore the science.

## Tech Stack

- **Frontend:** Vanilla HTML5, CSS3, JavaScript.
- **Typography:** Inter and Space Grotesk via Google Fonts.
- **Shaders:** Canvas-based radial gradients for organic glow effects.

## Quick Start

1. Clone the repository.
2. Open `index.html` in your browser.
